# NLP Audio Analysis System

## Overview

This project is an end-to-end Natural Language Processing (NLP) pipeline that transforms spoken audio into meaningful textual insights. The system automatically converts audio files into text, generates concise summaries, and extracts key topics from the content.

The project demonstrates the integration of speech recognition and NLP techniques to automate information extraction from unstructured audio data.

---

## Features

 **Speech-to-Text Conversion**

  * Converts audio files into text using OpenAI's Whisper model.
  * Supports multilingual transcription and robust speech recognition.

   **Text Summarization**

  * Generates concise summaries of lengthy transcripts.
  * Implements both:

    * Abstractive Summarization using BART.
    * Extractive Summarization using LSA.

  **Topic Extraction**

  * Identifies the dominant topics discussed in the audio.
  * Uses NLP techniques to highlight key themes and concepts.

   **Automated Pipeline**

  * Processes audio input and produces structured outputs with minimal user intervention.

## Results

The system provides three main outputs:

* **Transcribed Text:** Full textual representation of the audio.
* **Summary:** A concise overview of the main content.
* **Extracted Topics:** Keywords representing the dominant themes discussed.
